# PawStore

A pet products store built with React and Vite. Browse a catalog of quality products for your pets — beds, toys, accessories and more.

## Tech Stack

- **React** — UI and component structure
- **React Router** — client-side routing
- **Vite** — development server and bundler
- **CSS** — custom styles, no UI library

## Features

- Product catalog with responsive grid layout
- Product detail page
- Navigation with React Router
- Hover animations on product cards

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── assets/         # Images and SVGs
├── components/     # React components
├── data/           # JSON product data
└── styles/         # CSS files per component
```
